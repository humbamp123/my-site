# Wire Nut Size Calculator

A simple web application that helps determine the appropriate wire nut size and brand based on wire combinations.

## Features

- Support for multiple wire gauges (18-6 AWG)
- Dynamic wire entry addition/removal
- Comprehensive database of wire nuts from major brands:
  - Ideal Wing-Nut
  - 3M Scotchlok
  - Gardner Bender WingGard
  - King Innovation TwistOn
- Smart recommendations showing ideal matches and compatible alternatives
- Responsive design with modern UI
- Color-coded wire nut sizes for easy identification

## How to Use

Simply open `index.html` in any web browser. No installation or dependencies required!

1. Select wire gauge and enter quantity for each wire type
2. Click "Add Different Wire Gauge" if you need to mix different gauges
3. Click "Calculate Wire Nut Size" to see recommendations

## Publishing to GitHub Pages

To publish this application to GitHub Pages:

1. Go to your repository on GitHub: `https://github.com/humbamp123/wire-nuts`
2. Click on **Settings** (top menu)
3. Scroll down to the **Pages** section in the left sidebar
4. Under **Source**, select the branch: `claude/wire-nut-size-calculator-011CV1VJm5GxJJ7jYoBwYfN2`
5. Keep the folder as `/ (root)`
6. Click **Save**
7. Wait a few minutes for GitHub to build and deploy
8. Your site will be available at: `https://humbamp123.github.io/wire-nuts/`
