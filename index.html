The Wire Nut Calculator was built to solve a common problem on job sites—quickly determining the correct wire connector size for any combination of wires.

Whether you're working with residential 14/2 Romex or industrial-grade conductors, this tool provides instant recommendations from trusted brands including Ideal Wing-Nut, 3M Scotchlok, and more.

No more guessing, no more checking multiple spec sheets. Just select your wire gauges, enter quantities, and get accurate recommendations in seconds. Built by electricians, for electricians.
